# PartOfReader
一个H5 WEB小说阅读器的一部分，主要是显示每个章节的内容，浏览小说章节以及翻动页面
由于是移动端使用，所以为了减少对服务器的请求就将CSS+JS+HTML写在了一个页面内部。
